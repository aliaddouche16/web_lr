2. Усі веб -сторінки починаються з елемента html. Його також називають кореневим елементом, оскільки він знаходиться в корені дерева елементів, що складають веб -сторінку.

Щоб створити елемент html, пишеться початковий тег html, а потім закриваючий /html тег. Всі інші теги пишуться між цими двома.

3. Елемент head містить інформацію про веб-сторінку, але не вміст (вміст у body). Є багато елементів, які можна помістити всередину елемента head, наприклад:

  • Заголовок (title)
  • Посилання, яке можна використовувати для додавання таблиць стилів та значків на свою сторінку (link css)
  • Meta, для визначення таких речей, як набори символів, описи сторінок та ключові слова для пошукових систем.
  • Скрипт, для додавання коду JavaScript на сторінку.

4. Елемент заголовка містить заголовок сторінки. Заголовок відображається у вкладці браузера, а також у результатах пошукової системи та багатьох інших місцях.

Заголовок повинен коротко і точно описувати вміст сторінки. Слід надавати кожній сторінці сайту свій унікальний заголовок.

5. Елемент body з'являється після елемента head на сторінці. Він повинен містити весь вміст веб-сторінки: текст, зображення тощо. Усі веб-сторінки мають 1 єдиний елемент body, за винятком фреймів, які замість цього містять елементи фрейму. (фрейм - окремий HTML-документ, який сам чи разом з іншими документами відображений у вікні веб-переглядача.)

6. Заголовки дозволяють розділити вміст сторінки на розділи. Вони працюють так само, як заголовки та підзаголовки в книзі чи звіті.

HTML насправді підтримує 6 елементів заголовка: h1, h2, h3, h4, h5 і h6. h1 - для найважливіших заголовків, h2 - для менш важливих підзаголовків тощо. Зазвичай не потрібно використовувати більше ніж h1, h2 і h3, якщо ваша сторінка не дуже довга і складна.

7. Елемент p дозволяє створювати абзаци тексту. Більшість браузерів відображають абзаци з вертикальним проміжком між кожним абзацом, красиво розбиваючи текст.

Хоча можна створювати «абзаци» тексту, просто використовуючи теги, щоб вставляти порожні рядки між фрагментами тексту, краще замість цього використовувати елементи p. Він не тільки більш акуратний, але й дає браузерам та пошуковим системам краще уявлення про структуру вашої сторінки.

8. Один з найважливіших елементів веб-сторінки, елемент дозволяє створювати посилання на інший вміст.

Щоб створити посилання, слід обрати теги a та /a навколо вмісту, який ви хочете використовувати для посилання, і надати URL-адресу для посилання в атрибуті href тегу a.

9. Елемент img дозволяє вставляти зображення на веб-сторінки. Щоб вставити зображення, спочатку потрібно завантажити зображення на веб-сервер, а потім використовувати тег img для посилання на назву завантаженого файлу зображення.

Атрибут alt необхідний для всіх тегів img. Він використовується браузерами, які не відображають зображення, щоб дати альтернативний текст відвідувачу.

10. Елемент div - це загальний контейнер, який можна використовувати для додавання більшої структури вмісту сторінки. Наприклад, можна згрупувати кілька абзаців або заголовків, які мають подібну мету, всередині елемента div. Зазвичай елементи div використовуються для таких речей, як:

  • Верхні та нижні колонтитули сторінок
  • Стовпці вмісту та бічні панелі.
  • Виділені вікна в текстовому потоці.
  • Області сторінки з певною метою, наприклад, місця для реклами.
  • Галереї зображень.

Додавши атрибути класу та/або id до своїх елементів div, ви можете використовувати CSS для стилізації та розташування розділів.

11. Елемент span подібний до div тим, що використовується для додавання структури до вашого вмісту. Різниця в тому, що div є елементом на рівні блоку, тоді як span-це вбудований елемент:

Елементи на рівні блоків, такі як div, h1 і p,-це елементи, призначені для розміщення відносно великих або окремих блоків вмісту, таких як абзаци тексту. Елемент на рівні блоку завжди починається з нового рядка.

Вбудовані елементи, такі як span, a та img, призначені для розміщення менших фрагментів вмісту - наприклад, кількох слів чи речень - у великому блоці вмісту. Додавання вбудованого елемента не призводить до створення нового рядка. Елементи на рівні блоку можуть містити вбудовані елементи, але вбудовані елементи не можуть містити елементи на рівні блоків.
